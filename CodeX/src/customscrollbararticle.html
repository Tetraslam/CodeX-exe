Learn How To Make A Custom Scrollbar With CSS!

Using the code below, you can add your own custom scrollbar to any website! Just include it in the CSS or put it in a style tag on each page. In fact, we included some placeholder text to fill up the space just so you could see the gradient on our application's custom scrollbar. Try scrolling quickly and look at the colors change! (You'l only see it change if you're scrolling using a mouse button or trackpad, not if you click on the sidebar, in which case the colors will only invert.)

    ::-webkit-scrollbar {
  width: 15px;
  background-color: slategray;

}


::-webkit-scrollbar-track {
  box-shadow: inset 0 0 5px grey;
  border-radius: 10px;
}


::-webkit-scrollbar-thumb {
  background: rgb(16, 179, 103);
  border-radius: 10px;
}

The above code uses webkit, which is a web engine that powers applications, mostly on Unix-based systems.
